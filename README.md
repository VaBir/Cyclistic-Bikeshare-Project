 Cyclistic Bike-Share Analysis
**Google Data Analytics Capstone Project**

 Business Task
Determine how casual riders and annual members use Cyclistic bikes differently, in order to guide marketing strategies that convert casual riders into members. 
Αnalyzed 2023 Cyclistic bike-share data (5.8K bikes, 600 stations) to understand differences between the habits of casual users and annual members.

 Tools Used
- R (data cleaning)
- SQL (data exploration)
- Tableau (visual dashboards)
